# PopClip Software — Portable Text Selection Menu

## About PopClip Software

**PopClip Software** provides a complete popup menu that appears when you select text — copy, paste, cut, capitalize, search Google, translate, and more — for writers and editors who want instant text actions. **PopClip Software** combines selection detection, floating toolbar, and extension support into one cohesive interface. **PopClip Software** supports copying, pasting, search, translation, dictionary lookup, and note sending. **PopClip Software** includes customizable buttons, regex actions, and AppleScript extensions. **PopClip Software** enables users to perform text operations without reaching for keyboard shortcuts. **PopClip Software** is trusted by writers, editors, and anyone who works with text.

Capitalize button — transform case (uppercase, lowercase, title case)
Translate button — translate selected text
Extension support — Grammarly, ChatGPT, Dictionary, Notes, Reminders
